## Icon Requirements

- **Format**: SVG preferred
- **Size**: 24x24 viewBox recommended
- **Style**: Should match existing Tabler/Streamline icon style
- **Colors**: Should work with stroke="#000000" for consistency
- **License**: Must be compatible with project licensing

## Sources

- Official service websites for branded icons
- [Tabler Icons](https://tabler-icons.io/) for generic alternatives
- [Streamline Icons](https://streamlinehq.com/) for premium alternatives
- Custom creation if official assets unavailable
